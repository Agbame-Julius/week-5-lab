Amazon ECS

Elastic Container Service

Amazon Elastic Container Service (ECS) is a fully managed container orchestration service that makes it easy to deploy, manage, and scale containerized applications.

Key Features:
  • Fully managed container orchestration
  • AWS Fargate integration for serverless compute
  • Deep integration with AWS services
  • Simplified cluster management
  • Automated container deployment

Amazon ECR

Elastic Container Registry

Amazon Elastic Container Registry (ECR) is a fully managed container registry that makes it easy to store, manage, and deploy container images.

Key Features:
  • Secure private container image storage
  • Integrated with ECS and EKS
  • Lifecycle policies to manage images
  • Image scanning for vulnerabilities
  • Cross-region replication
